# Welcome to IamThiago Storybook

This is the Storybook for **IamThiago**, a minimalist developer portfolio.

## Structure

The stories are organized into two main categories:

- **UI**: Low-level, reusable components (mostly from Shadcn UI).
- **Components**: Higher-level application components.

## Tech Stack

- **Next.js**: Framework
- **Tailwind CSS**: Styling
- **Shadcn UI**: Base UI components
- **Framer Motion**: Animations
- **Lucide React**: Icons
- **i18n**: Internationalization support

## Global Providers

This Storybook is configured with the following global providers:

- `I18nProvider`: For translation support.
- `ThemeProvider`: For light/dark mode support.
- `Global CSS`: Tailwind CSS and custom variables.

## How to use

You can explore the components in the sidebar. Use the **Controls** tab to interact with component props and see how they behave.
